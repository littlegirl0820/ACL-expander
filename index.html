Browser ACL Expander

Unofficial tool. Expand official C++ ACL headers, Python ACL modules, or Rust ac-library-rs code entirely in your browser.

Input

C++ / Python / Rust の ACL コードを貼り付けます。

Output

展開結果をコピーまたは保存できます。

Unofficial fan-made tool. All expansion is done client-side in the browser. C++ AtCoder Library is distributed under CC0-1.0, `ac-library-python` is distributed under MIT, and `ac-library-rs` is distributed under CC0-1.0. Sources: atcoder/ac-library / not522/ac-library-python / rust-lang-ja/ac-library-rs
